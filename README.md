## Getting Started

### Installation

#### Frontend
Open a terminal in the root folder of this project.

```sh
cd InimcoTestFrontend/inimco-test-frontend-app
npm install npm
npm start
```

#### Backend
Open the solution in your favorite IDE (Rider, Visual Studio...) and run the InimcoBackend project.
